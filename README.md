## Comment lancer le bot ?

Allez dans `config.js` et changez le [Token](https://discord.com/developers), puis incérer votre lien [Database](https://www.mongodb.com/atlas)
```json
{
    "Token": "TOKEN",
    "Database": "URL DATABASE"
}
```
